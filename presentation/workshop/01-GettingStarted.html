Getting Started with LoopBack

Prerequisites

  1. Node.js & npm Current Version: v0.10.26

  2. strong-cli

  3. git command-line tool

  4. Xcode Xcode 4.6 or higher ( requires Mac OSX ): for Native iOS mobile app

  5. Eclipse Android Developement Tools Eclipse Android Developement Tools, ADT : for Native Android mobile app

  6. At least one Android or virtual Android device, refer to AVD Manager for more information.

Installing slc

Install strong-cli command line utility from npm

npm install -g strong-cli
slc --help

Start your project

Create a folder for our project:


        mkdir workshop-strongloop-loopback
        cd workshop-strongloop-loopback

Create your LoopBack Node.js project

Create your LoopBack Node.js server with the slc command: 


			slc lb project node-server
      cd node-server

Strong Ops

Register and configure Strong Ops 

  slc strongops

Give it a run

Start your loopback Node.js server: 

 slc run app.js

Give it a run in the debugger

Debug your loopback Node.js server: 

  slc debug app.js

slc example

Take the easy way 

 
	$ slc example
	cd sls-sample-app
	…

	$slc lb datasource mongo --connector mongodb
	$slc strongops
	$slc run app.js
	…
	$node bin/create-load.js
